#  Cliente de API OpenWeather em C

##  Objetivo
Programa em C que consulta a API pública **OpenWeather** para obter a **temperatura atual** de uma cidade informada pelo usuário.  
O resultado é salvo em um arquivo JSON e exibido no terminal.

---

##  Compilação e Execução
**Compilação:**
```bash
gcc main.c -o main.exe
.\main.exe
```

### Ajuste de including para o terminal do Windows: "chcp 1252" (executar no terminal antes de rodar).
